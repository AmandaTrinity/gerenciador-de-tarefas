# Gerenciador de Tarefas com React, Vite e Tailwind CSS

Um gerenciador de tarefas simples criado com **React**, **Vite** e **Tailwind CSS**. Permite adicionar, editar, remover e marcar tarefas como concluídas.

## Tecnologias

- **React**
- **Vite**
- **Tailwind CSS**

## Instalação

1. Clone o repositório:
- git clone https://github.com/seu-usuario/nome-do-repositorio.git

2. Instale as dependências:
- npm install

3. Inicie o servidor:

- npm run dev
- Acesse no navegador: http://localhost:3000.
